# Privacy and Security

This extension uses Google's Gemini API to generate responses. Chat messages are stored in Firestore and sent to Google's servers for processing.

## Data Collection

- Chat messages and metadata are stored in Firestore
- Messages are sent to Google's Gemini API
- No additional data is collected

## Security

- Uses your Firebase project's security rules
- API keys are stored securely
- Follows Firebase extension security best practices

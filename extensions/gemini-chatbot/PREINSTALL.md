# Pre-Installation Instructions

## Prerequisites

- Firebase project with Firestore enabled
- Google Gemini API key
- Billing enabled on Firebase project

## Required APIs

This extension requires the following Google APIs to be enabled:

- Cloud Firestore API
- Gemini API

## Costs

- Firebase Extensions: Free
- Gemini API: Pay-per-use (see [Gemini Pricing](https://cloud.google.com/vertex-ai/pricing))
- Firestore: Usage-based pricing
